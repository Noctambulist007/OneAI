## 🚀 Installation

### Prerequisites
- Flutter 3.3.0 or higher
- Dart SDK 3.0.0 or higher
- Google Cloud API key with Gemini AI enabled
- Firebase project

### Setup Steps

```bash
# Clone this repository
$ git clone https://github.com/yourusername/oneai.git

# Go into the repository
$ cd oneai

# Install dependencies
$ flutter pub get

# Create .env file for API keys
$ touch .env

# Add your Gemini API key to .env file
# GEMINI_API_KEY=your_api_key_here

# Setup Firebase
$ flutter pub add firebase_core
$ flutter pub add firebase_auth
$ flutter pub add cloud_firestore
$ flutterfire configure

# Run the app
$ flutter run
```

### Configuration
Create a `.env` file in the project root with your API keys:

```
GEMINI_API_KEY=your_gemini_api_key_here
```

## 💻 Code Examples

### Processing Text with Gemini AI

```dart
// Sample code for Gemini AI text processing
Future<String> processTextWithAI(String inputText, String action, String language) async {
  try {
    final model = GenerativeModel(
      model: 'gemini-pro',
      apiKey: dotenv.env['GEMINI_API_KEY']!,
    );
    
    // Use our PromptHelper to format the request based on action and language
    final prompt = PromptHelper.getPrompt(action, inputText, language);
    final content = [Content.text(prompt)];
    final response = await model.generateContent(content);
    
    return response.text ?? 'No response generated';
  } catch (e) {
    return 'Error processing text: $e';
  }
}
```

### Image Analysis with Gemini Vision

```dart
// Sample code for Gemini vision image processing
Future<String> analyzeImageWithAI(File imageFile, String feature) async {
  try {
    final model = GenerativeModel(
      model: 'gemini-pro-vision',
      apiKey: dotenv.env['GEMINI_API_KEY']!,
    );
    
    // Get the appropriate prompt for the selected feature
    final prompt = prompts[feature] ?? 'Describe this image in detail:';
    
    // Convert image to base64
    final bytes = await imageFile.readAsBytes();
    final base64Image = base64Encode(bytes);
    
    final content = [
      Content.text(prompt),
      Content.image(base64Image)
    ];
    
    final response = await model.generateContent(content);
    return response.text ?? 'No analysis generated';
  } catch (e) {
    return 'Error analyzing image: $e';
  }
}
```
